## 执行命令模板

```bash
python scripts/run_pipeline.py \
  --symbol 600519.SH \
  --strategy <strategy_name> \
  --start 2020-01-01 \
  --end 2026-04-26 \
```

## 计划说明

- 本计划故意固定标的和时间范围，只变策略，避免把不同股票的波动混进对比结果。
- 选择五个经典策略，是为了覆盖趋势、震荡、突破、反转四类常见逻辑。
- 若五次均成功，后续可继续做第二轮对比，例如加入 `momentum` 或换成多标的批量比较。
